## Simple Rating Stars

A display-only React component to render basic star-ratings.

`npm install simple-rating-stars`

Use it like this:

```
  <Stars
    stars={3}
    outOf={5}
    activeColor={'#385269'}
    inactiveColor={'#FFF'}
    stroke={'#000'}
  />
```
